## 一、数组基础

数组是**一排连续的同类型变量**。

```cpp
int arr[5] = {10, 20, 30, 40, 50};
```

**内存布局：**
```
arr ──→ [10][20][30][40][50]
         [0]  [1]  [2]  [3]  [4]   ← 下标从 0 开始
```

**遍历数组：**
```cpp
for (int i = 0; i < 5; i++) {
    cout << arr[i] << endl;
}
```

## 二、数组名就是指针

```cpp
arr  == &arr[0]     // 数组名 = 首元素地址
*arr == arr[0]      // 解引用数组名 = 第一个元素
```

**通用规律：**
```cpp
arr[i] == *(arr + i)
```

## 三、用指针遍历数组

```cpp
int arr[5] = {5, 15, 25, 35, 45};
int* p = arr;              // p 指向数组首地址

for (int i = 0; i < 5; i++) {
    cout << *p << endl;    // 解引用取值
    p++;                   // 指向下一个元素
}
```

**指针移动的本质：** `p++` 不是加 1 个字节，而是加 `sizeof(int)` = 4 字节，跳到下一个元素。

## 四、数组 vs 指针

```cpp
int arr[5];     // arr 是固定在墙上的地址
int* p;         // p 是可以改写的地址纸条

p = arr;        // ✅ p 可以指向数组
// arr = p;     // ❌ 数组名不能改指向
```

## 五、二级指针（指向指针的指针）

```
int x = 42;
int* p = &x;    // p 存 x 的地址
int** pp = &p;  // pp 存 p 的地址
```

```
变量     值           地址
─────────────────────────────────
x        42           0x100
p        0x100        0x200
pp       0x200        0x300
```

**解引用：**
- `*pp`  → 得到 p 的值（&x，即 0x100）
- `**pp` → 得到 x 的值（42）

**实战：二级指针实现交换**
```cpp
void swapWithPP(int** a, int** b) {
    int temp = **a;     // 两次解引用取到 x 的值
    **a = **b;
    **b = temp;
}

int main() {
    int x = 5, y = 10;
    int* pa = &x;
    int* pb = &y;
    swapWithPP(&pa, &pb);   // 传指针的地址
    cout << x << " " << y;  // 输出：10 5
}
```

## 六、本章小结

| 概念 | 写法 | 含义 |
|------|------|------|
| 数组名 | `arr` | 首地址（&arr[0]）|
| 下标访问 | `arr[i]` | 等价于 `*(arr + i)` |
| 指针遍历 | `int* p = arr; p++` | 每次移动 sizeof(T) 字节 |
| 一级指针 | `int* p` | 存变量的地址 |
| 二级指针 | `int** pp` | 存指针的地址 |
