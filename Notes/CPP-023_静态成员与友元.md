## 一、静态成员（static）

### 为什么需要静态成员？

需要**所有对象共享**的数据，比如统计一共创建了多少个对象。

```cpp
class Student {
    string name;
    static int count;   // 声明：属于类，所有对象共享
public:
    Student(string n) : name(n) { count++; }
    ~Student() { count--; }
    static int getCount() { return count; }  // 静态成员函数
};

// 定义并初始化（必须在类外面！）
int Student::count = 0;
```

### 关键规则

| 规则 | 说明 |
|------|------|
| 属于谁 | 属于**类**，不属于某个对象 |
| 如何访问 | `类名::成员`（不需要创建对象）|
| 存储位置 | 必须在**类外面**单独定义一次 |
| 静态函数 | 没有 `this` 指针，不能访问非静态成员 |

```cpp
cout << Student::getCount();  // ✅ 无需对象
cout << stu.getCount();       // ✅ 也可以（但不推荐）
```

### 静态函数为什么不能访问普通成员？

普通成员变量依附于具体对象，静态函数没有 `this`，不知道访问哪个对象的数据。

## 二、友元（friend）

### 为什么需要友元？

外部函数或类需要访问某个类的 `private` 成员时。

最常见场景——**运算符重载**，比如 `operator<<` 需要访问私有成员来输出。

### 用法

```cpp
class Student {
    friend ostream& operator<<(ostream& os, const Student& stu);
private:
    string name;
    int score;
public:
    Student(string n, int s) : name(n), score(s) {}
};

ostream& operator<<(ostream& os, const Student& stu) {
    os << stu.name << ": " << stu.score;
    return os;
}

int main() {
    Student stu("小明", 85);
    cout << stu << endl;   // 小明: 85
}
```

### 友元的规则

| 规则 | 说明 |
|------|------|
| 写在类内部 | `friend` 声明放在类里，哪部分都行 |
| 不是成员函数 | 它是外部函数，只是被赋予了访问权限 |
| 位置无关 | `private` 或 `public` 区域效果一样 |
| 不能继承 | 父类的友元不会自动成为子类的友元 |

## 三、运算符重载入门

### 本质

运算符重载就是写一个名字叫 `operator@` 的函数，`@` 是你想重载的运算符。

### 什么时候用哪种写法？

| 运算符 | 左操作数 | 右操作数 | 写法 |
|--------|---------|---------|------|
| `<<` `>>` | cout/cin | 你的对象 | **外部函数 + 友元** |
| `>` `<` `==` `+` `-` | 你的对象 | 你的对象 | **成员函数** |
| `[]` | 你的对象 | 下标 | **成员函数** |

### 作为成员函数（适合 `>`、`+` 等）

```cpp
bool operator>(const Pet& other) const {
    return price > other.price;  // this->price > other.price
}
// 调用：a > b → a.operator>(b)
```

### 作为外部函数（适合 `<<`、`>>` 等）

```cpp
ostream& operator<<(ostream& os, const Pet& pet) {
    os << pet.name << " " << pet.price;
    return os;
}
// 调用：cout << pet
```

## 四、`&` 符号的三种含义（复习）

```cpp
int x = 42;
int* p = &x;      // ① 取地址符 —— 表达式里
int &r = x;       // ② 声明引用 —— 类型后面
int a = b & c;    // ③ 按位与运算符
```

## 五、快速参考

| 概念 | 一句话 |
|------|--------|
| `static` 成员变量 | 属于类，所有对象共享，类外定义 |
| `static` 成员函数 | 属于类，没有 `this`，不能访问非静态 |
| `friend` | 允许外部函数访问私有成员 |
| `operator@` | 重载运算符，让自定义类型支持原生操作符 |
