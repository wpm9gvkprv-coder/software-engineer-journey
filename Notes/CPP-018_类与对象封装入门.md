# CPP-018：类与对象（封装入门）

**日期：** 2026-07-17  
**位置：** 阶段三 — 面向对象编程

---

## 一、为什么需要类？

传统方式：数据和函数分离，谁都可以修改数据

```cpp
int scores[100];
int count = 0;

void addScore(int s) {
    scores[count] = s;
    count++;
}

// 问题：不小心写个 count = -1，数据就全乱了
```

类解决了这个问题：**把数据和操作数据的方法打包在一起，并控制外界访问权限。**

## 二、定义一个类

```cpp
class Student {
private:                // 私有——外部不能直接访问
    string name;
    int score;

public:                 // 公有——对外提供的接口
    void setName(string n) { name = n; }
    string getName() { return name; }
    void setScore(int s) {
        if (s >= 0 && s <= 100) {    // 校验保护
            score = s;
        }
    }
    int getScore() { return score; }
};
```

### 访问控制

| 关键字 | 作用 |
|--------|------|
| `private` | 类内部可访问，外部不可访问 |
| `public` | 谁都可以访问 |

**封装的意义：** 通过 `private` 隐藏数据，通过 `public` 接口控制访问方式，保护数据完整性。

## 三、使用类（创建对象）

```cpp
int main() {
    Student stu;
    stu.setName("小明");
    stu.setScore(85);
    
    // stu.score = 100;  ❌ 编译错误！score 是 private
    cout << stu.getName() << "的分数是" << stu.getScore() << endl;
}
```

## 四、构造函数（Constructor）

**作用：** 创建对象时自动初始化，不需要分两步走。

**规则：**
- 函数名必须和类名相同
- 没有返回类型（连 `void` 都不能写）
- 创建对象时自动调用
- 支持重载（多个版本的构造函数）

```cpp
class Student {
public:
    // 有参构造函数
    Student(string n, int s) {
        name = n;
        score = (s >= 0 && s <= 100) ? s : 0;
    }
    
    // 无参构造函数（重载）
    Student() {
        name = "未命名";
        score = 0;
    }
};

int main() {
    Student a;                // 调用无参构造
    Student b("小明", 85);    // 调用有参构造
}
```

### ⚠️ 注意
一旦手写了任意一个构造函数，编译器就不再自动生成默认无参构造函数。如果还需要无参构造，必须**手动写一个重载版本**。

## 五、封装的好处

```cpp
void setScore(int s) {
    if (s >= 0 && s <= 100) {    // ① 可以加校验
        score = s;
    }
}
```

1. **数据保护** —— 不允许外部随意修改内部数据
2. **校验逻辑** —— setter 中可以加入合法性检查
3. **接口稳定** —— 内部实现改了，对外接口不变

## 六、易错点

```cpp
class Student {
    privite:    // ❌ 拼写错误！是 private 不是 privite
```

```cpp
void getScore(int s) {   // ❌ 函数名是 get，却在做 set 的事
    score = s;
}
```

给自己声明规则：**set 开头的方法用于设置，get 开头的方法用于获取。**

## 七、本章小结

| 概念 | 说明 |
|------|------|
| 类（class） | 数据和函数的打包 |
| 对象 | 类的具体实例 |
| private | 私有成员，外部不可访问 |
| public | 公有接口，外部可调用 |
| 封装 | 隐藏数据，暴露接口 |
| 构造函数 | 创建时自动初始化 |
| 重载 | 同名函数，不同参数 |
