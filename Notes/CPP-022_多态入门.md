# CPP-022：多态入门

**日期：** 2026-07-19  
**位置：** 阶段三 — 面向对象编程

---

## 一、为什么需要多态？

```cpp
class Animal {
public:
    void speak() { cout << "动物叫" << endl; }
};

class Dog : public Animal {
public:
    void speak() { cout << "汪汪！" << endl; }
};

int main() {
    Animal* p = new Dog();
    p->speak();   // ❌ 输出"动物叫"——明明指向的是狗！
}
```

**问题：** 编译器只看指针类型（`Animal*`），不看实际指向的对象（`Dog`）。

**多态解决：** 运行时根据**实际对象类型**调用对应的方法。

---

## 二、虚函数（virtual）

在父类函数前加 `virtual`，子类重写：

```cpp
class Animal {
public:
    virtual void speak() { cout << "动物叫" << endl; }
};

class Dog : public Animal {
public:
    void speak() override { cout << "汪汪！" << endl; }
};

int main() {
    Animal* p = new Dog();
    p->speak();   // ✅ 输出"汪汪！"
    delete p;
}
```

| 关键字 | 位置 | 作用 |
|--------|------|------|
| `virtual` | 父类函数前 | 启用多态 |
| `override` | 子类重写函数后 | （C++11）明确表示重写父类虚函数，写错名会报错 |

## 三、统一接口

多态真正的威力——**写一次调用代码，对所有子类通用**：

```cpp
void makeItSpeak(Animal* p) {
    p->speak();   // 不管传来 Dog 还是 Cat，都能正确调用
}

int main() {
    Dog dog;
    Cat cat;
    makeItSpeak(&dog);   // 汪汪！
    makeItSpeak(&cat);   // 喵~
}
```

**扩展新动物时不需要修改 `makeItSpeak`** → 对扩展开放，对修改封闭（开闭原则）。

## 四、虚析构函数（Virtual Destructor）

**关键规则：** 凡是声明了虚函数的类，析构函数也必须是 virtual 的。

```cpp
class Animal {
public:
    virtual ~Animal() = default;   // 虚析构
    virtual void speak() = 0;
};

class Dog : public Animal {
public:
    ~Dog() override { /* 释放 Dog 的资源 */ }
    void speak() override { cout << "汪汪！" << endl; }
};

int main() {
    Animal* p = new Dog();
    delete p;   // ✅ 先调用 Dog 析构，再调用 Animal 析构
}
```

**为什么？** 否则 `delete` 父类指针时，不会调用子类析构，导致内存泄漏。

## 五、纯虚函数与抽象类

```cpp
class Animal {
public:
    virtual void speak() = 0;   // 纯虚函数：没有函数体，= 0 结尾
    virtual ~Animal() = default;
};
```

| 特性 | 说明 |
|------|------|
| 纯虚函数 | `= 0` 结尾，没有函数体 |
| 抽象类 | 包含纯虚函数的类 |
| 能否实例化 | ❌ 不能：`Animal a;` 编译错误 |
| 强制要求 | 子类必须实现所有纯虚函数才能实例化 |

---

## 六、面向对象三大基石

| 概念 | 作用 |
|------|------|
| **封装** | `private` 隐藏数据，`public` 暴露接口 |
| **继承** | 提取共性到基类，派生类扩展 |
| **多态** | 同一接口，不同行为（`virtual`） |

---

## 七、完整示例

```cpp
class Animal {
protected:
    string name;
    int age;
public:
    Animal(string n, int a) : name(n), age(a) {}
    virtual void speak() = 0;     // 纯虚函数
    virtual ~Animal() = default;   // 虚析构
};

class Dog : public Animal {
public:
    Dog(string n, int a) : Animal(n, a) {}
    void speak() override { cout << "汪汪！" << endl; }
};

class Cat : public Animal {
public:
    Cat(string n, int a) : Animal(n, a) {}
    void speak() override { cout << "喵~" << endl; }
};

void makeItSpeak(Animal* p) {
    p->speak();
}

int main() {
    Dog dog("旺财", 3);
    Cat cat("咪咪", 2);
    makeItSpeak(&dog);   // 汪汪！
    makeItSpeak(&cat);   // 喵~
}
```
